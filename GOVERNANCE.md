# Captain Culinary Pro Hub governance

Captain Culinary Pro Hub is a BCA product that consumes shared platform services; it is not the BCA operating system or the source of truth for company-wide architecture.

The Open-LLM-VTuber evaluation must remain isolated, pinned, and replaceable. Third-party frontend code and sample character assets may not enter the product without explicit license approval.

Repository import strategy, shared Live Avatar Agent Engine boundaries, persistent memory, device permissions, agent tools, and Continuity Mode synchronization are architecture-level decisions routed through Conrad/EXPO.
